# PharmaFind Frontend

The user interface for the PharmaFind platform, built with React and Vite.

## Features
- **Pharmacy Discovery:** Interactive search and filtering of pharmacies.
- **Inventory Management:** Modern dashboard for pharmacy owners.
- **Auth Flow:** Secure login/signup for consumers and pharmacies.
- **Reservation System:** Integrated checkout with Stripe.

## Development
```bash
npm install
npm run dev
```
