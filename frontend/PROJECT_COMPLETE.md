## 🔐 Security & Architecture

```
┌─────────────────────────────────────────────────────────────┐
│  PROTECTED ROUTES                                           │
├─────────────────────────────────────────────────────────────┤
│  All pharmacy routes wrapped with:                          │
│  <ProtectedRoute requirePharmacy>                           │
│                                                             │
│  Authentication Flow:                                       │
│  Login → AuthContext → Token Storage → Route Access        │
│                                                             │
│  Authorization:                                             │
│  • Pharmacy role required                                   │
│  • Redirects non-pharmacy users                             │
│  • Protects sensitive data                                  │
└─────────────────────────────────────────────────────────────┘
```
